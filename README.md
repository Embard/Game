# Photo Runner

Мини-игра в стиле раннера на чистых **HTML + CSS + JS + Canvas**.

## Структура

- `index.html` — разметка и элементы управления.
- `style.css` — оформление UI и адаптивность.
- `game.js` — логика игры, физика, отрисовка.
- `assets/player-photo.jpg` — фото персонажа.

## Запуск

1. Из корня проекта запустите локальный сервер:

```bash
python3 -m http.server 8000
```

2. Откройте в браузере: `http://localhost:8000`

> Можно открыть и просто `index.html`, но через локальный сервер загрузка ресурсов стабильнее.

## Использование фото персонажа

По умолчанию используется файл `assets/player-photo.jpg`.

Если нужно заменить персонажа:
1. Положите новое фото в `assets/player-photo.jpg`.
2. Перезагрузите страницу.
3. При необходимости подстройте `CONFIG.photo` в `game.js` (`focusX`, `focusY`, `zoom`).

Если фото не загрузится, игра использует встроенный fallback-персонаж.
